# Thala 7 Sneaker Store

Thala 7 is a polished multi-page sneaker storefront concept built with HTML, CSS, and JavaScript. The project focuses on strong visual hierarchy, responsive layout decisions, and a clean eCommerce user journey across landing, product, cart, and checkout pages.

## Portfolio Highlights

- Premium hero section with layered backgrounds, typography contrast, and clear call-to-action flow
- Consistent branding system shared across all pages
- Responsive navigation with mobile menu interaction
- Product gallery with thumbnail switching
- Clean cart and checkout layouts designed to feel production-inspired
- Resume-friendly front-end case study for static UI implementation

## Tech Stack

- HTML5
- CSS3
- JavaScript
- Responsive design with modern layout patterns

## How to Run

1. Clone or download the project.
2. Open [index.html](./index.html) in your browser.
3. Navigate through the product, cart, and checkout pages.
